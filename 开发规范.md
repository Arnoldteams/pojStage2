# commit

标签：

- feat：新功能的开发
- fix：bug的修复
- docs：文档格式的改动
- style：代码格式改变
- refactor：对已有的功能进行重构
- perf：性能优化
- test：增加测试
- build：改变了build工具
- revert：撤销上一次的commit提交
- chore：构建过程或辅助工具的变动

举例：

- git commit -m "yyf，0625，fix: 修复购物车数据不匹配问题"
- git commit -m "yyf，0625，feat：新增购物车功能"
